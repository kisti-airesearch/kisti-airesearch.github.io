# Large-scale AI Research Center @ KISTI

Website of the **Large-scale AI Research Center**, Division of AI and Data,
Korea Institute of Science and Technology Information (KISTI).

Live at: https://kisti-airesearch.github.io

## Structure

| File | Page |
| --- | --- |
| `index.html` | Home — about, research highlights, KONI models, news, contact |
| `research.html` | Research areas in detail |
| `people.html` | Center members |
| `publications.html` | Papers, open models, and datasets |
| `assets/css/style.css` | All styles |
| `assets/js/main.js` | Mobile navigation toggle |
| `images/` | KISTI logo and KION mascot |

## How to update

The site is plain HTML/CSS — no build step. Edit the HTML files and push to `main`;
GitHub Pages publishes automatically.

- **Add news**: edit the `News` section in `index.html` (copy an `<li>` item).
- **Add a member**: copy a `<div class="person">` block in `people.html` (instructions are in a comment there).
- **Add a publication**: copy an `<li>` block in `publications.html` under the right year.

Placeholder text to replace is marked with `[square brackets]` or `[Sample]`.
